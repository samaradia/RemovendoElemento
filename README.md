# Removendo Elementos de uma Lista em Java

Exemplo simples de utilização da classe `ArrayList` para adicionar e remover elementos de uma lista.

## Funcionalidades

- Criar uma lista
- Adicionar estudantes
- Remover um estudante
- Exibir a lista antes e depois da remoção

## Como executar

1. Compile o arquivo:
   ```bash
   javac Main.java
   ```

2. Execute:
   ```bash
   java Main
   ```

## Saída esperada

```text
lista inicial: [Joana, Lucas, Pedro, Antônio]
Lista após a exclusão: [Joana, Lucas, Antônio]
```
